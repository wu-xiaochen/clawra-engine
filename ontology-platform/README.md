# ontology-platform

垂直领域可信AI推理引擎平台

## 核心特性

- 🔥 消除AI幻觉 - 每步推理有据可查
- ⚡ 可解释推理 - 透明推理过程
- 📚 领域知识库 - 燃气工程、采购供应链、安全生产、金融风控
- 🏢 企业级可靠 - 可审计、可追溯

## 已覆盖领域

- ✅ 燃气工程
- ✅ 采购供应链
- ✅ 安全生产
- ✅ 金融风控
- ⏳ 医疗健康

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 运行API服务
python src/api.py

# 测试推理
python examples/demo.py
```
