# 约会领域本体设计

## 概述

本文档定义约会（Dating）领域的核心本体模型，用于指导婚恋平台、智能匹配系统、约会助手等应用的知识库构建和智能化服务。

---

## 一、概念定义 (Concepts)

### 1.1 核心实体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 用户 | User | 约会平台的注册用户 | 用户ID、昵称、年龄、性别、地理位置、兴趣爱好 |
| 个人资料 | Profile | 用户的详细资料信息 | 头像、简介、职业、学历、身高、收入、婚姻状况 |
| 约会对象 | Date | 约会的另一方参与者 | 对象ID、匹配度、互动状态 |
| 匹配 | Match | 双方互相喜欢的配对 | 匹配ID、匹配时间、匹配来源、状态 |
| 约会 | Date | 两人见面的社交活动 | 约会ID、日期时间、地点、类型、状态 |
| 约会场所 | Venue | 约会的地点 | 场所ID、名称、类型、环境、消费水平 |
| 约会活动 | Activity | 约会中进行的具体活动 | 活动ID、名称、类型、适合阶段、互动程度 |
| 消息 | Message | 用户之间的沟通内容 | 消息ID、发送者、接收者、内容、时间、已读状态 |
| 关系状态 | Relationship Status | 两人之间的情感状态 | 状态类型、开始时间、持续时长 |
| 偏好 | Preference | 用户对约会对象的期望 | 年龄范围、身高范围、学历、收入、兴趣标签 |

## 三、规律 (Laws)

### 3.1 匹配算法核心规律

**相似性匹配定律**

| 匹配维度 | 权重 | 规律描述 |
|----------|------|----------|
| 地理位置 | 15% | 距离越近，匹配成功率越高（最佳范围：5-20km） |
| 年龄匹配 | 10% | 年龄差在3-5岁内匹配度最高 |
| 兴趣爱好 | 25% | 共同兴趣数量与匹配满意度正相关 |
| 价值观 | 30% | 核心价值观契合是长期关系的关键 |
| 生活方式 | 20% | 生活节奏相似度影响相处舒适度 |

**匹配度计算公式**:
```
匹配度 = Σ(维度权重 × 维度相似度) × 活跃度系数 × 互惠系数
```

### 3.2 约会成功率规律

**初次约会黄金法则**

| 因素 | 影响 | 最佳实践 |
|------|------|----------|
| 约会时长 | 成功率与时长呈倒U型 | 1.5-2.5小时最佳 |
| 约会时间 | 周末下午 > 工作日晚间 | 周六下午2-5点 |
| 场所选择 | 半公开场所成功率最高 | 咖啡厅、餐厅、公园 |
| 主动程度 | 双方主动度平衡更易成功 | 轮流主动 |
| 话题深度 | 中等深度话题最适宜 | 避免过于私密或表面 |

**约会阶段成功率统计**:
```
第一次约会成功率: 35-45%
第二次约会邀请率: 60-70%
第三次约会确定关系率: 40-50%
五次约会后稳定交往率: 70-80%
```

### 3.3 消息互动规律

**回复率时间规律**

| 消息发送时间 | 平均回复率 | 最佳时段 |
|--------------|------------|----------|
| 早上7-9点 | 45% | 快速回复高发期 |
| 上午9-12点 | 35% | 工作时间，回复较慢 |
| 下午12-14点 | 50% | 午休时间，回复率高 |
| 下午14-18点 | 40% | 工作间隙 |
| 晚上18-21点 | 55% | 黄金时段，最高 |
| 晚上21-23点 | 48% | 睡前浏览 |
| 深夜23点后 | 30% | 休息时间，不推荐 |

**消息长度与回复率**:
```
过短 (<5字): 回复率 35%
适中 (10-30字): 回复率 65%
过长 (>100字): 回复率 40%
```

### 3.4 关系发展规律

**关系发展时间线**

| 阶段 | 平均时长 | 关键里程碑 |
|------|----------|------------|
| 匹配-首次聊天 | 24-72小时 | 首日互动很重要 |
| 聊天-交换联系方式 | 3-7天 | 微信/电话交换 |
| 联系方式-首次约会 | 7-14天 | 首次见面 |
| 首次约会-第二次 | 2-5天 | 反馈及时性 |
| 多次约会-确定关系 | 2-6周 | 表白/确认 |
| 恋爱-同居 | 6-18个月 | 考虑现实因素 |

**关系进展标志信号**:
- 愿意主动发消息：兴趣指标 +
- 消息内容逐渐深入：信任建立 +
- 同意快速见面：意愿强烈 +
- 愿意分享日常：亲密感 +
- 主动肢体接触：关系升级信号

## 四、业务规则 (Rules)

### 4.1 用户匹配规则

**匹配准入条件**

| 条件类型 | 具体要求 |
|----------|----------|
| 基础条件 | 双方均已完成资料认证 |
| 年龄限制 | 男方22-60岁，女方20-55岁 |
| 地区限制 | 同城或邻近城市 |
| 状态限制 | 双方均为单身状态 |
| 互惠规则 | 双方至少一方对另一方有好感 |

**匹配推荐优先级**

| 优先级 | 推荐规则 |
|--------|----------|
| P0 | 高度匹配（匹配度>85%）+ 活跃用户 |
| P1 | 高匹配度（70-85%）+ 最近登录 |
| P2 | 中等匹配度（50-70%）+ 共同兴趣多 |
| P3 | 扩展推荐（匹配度<50%）+ 新用户优先 |

### 4.2 消息互动规则

**消息发送规则**

| 规则类型 | 具体内容 |
|----------|----------|
| 匹配后首条消息 | 24小时内发送最佳，超过7天未发默认放弃 |
| 消息频率 | 首日1-3条为宜，避免连续轰炸 |
| 敏感词过滤 | 自动过滤广告、诈骗、违规内容 |
| 敏感话题 | 初次聊天避免收入、婚史、前任等话题 |
| 互动保护 | 连续3天未回复可提醒一次 |

**消息内容规范**

| 推荐话题 | 避免话题 |
|----------|----------|
| 兴趣爱好 | 收入存款 |
| 旅行经历 | 前任情况 |
| 美食探索 | 身体状况 |
| 电影音乐 | 政治宗教 |
| 职业趣事 | 相亲动机 |
| 童年往事 | 催婚压力 |

### 4.3 约会安排规则

**首次约会规范**

| 要素 | 建议规范 |
|------|----------|
| 时间 | 周末下午2-5点为宜 |
| 时长 | 1.5-2小时 |
| 场所 | 公共场所、交通便利 |
| 消费 | 建议AA制或轮流请客 |
| 安全 | 告知亲友行程、保持警惕 |

**约会场所选择规则**

| 关系阶段 | 推荐场所类型 | 不推荐 |
|----------|--------------|--------|
| 初次约会 | 咖啡厅、书店、公园 | 密室、电影院 |
| 第二次约会 | 餐厅、展览、动物园 | 家中、私人影院 |
| 熟悉后 | 徒步、做饭、周边游 | - |
| 确定关系 | 正式餐厅、浪漫场所 | - |

### 4.4 关系进展规则

**升级关系触发条件**

| 关系升级 | 触发条件 | 确认方式 |
|----------|----------|----------|
| 聊天→约会 | 聊天超过50条 + 3天以上 | 主动邀约 |
| 约会→暧昧 | 约会2次以上 + 互动积极 | 肢体接触 |
| 暧昧→表白 | 相处超过1个月 + 明确暗示 | 语言确认 |
| 交往→同居 | 交往6个月以上 + 共同意愿 | 共同生活 |

**关系降级/终止规则**

| 情况 | 处理方式 |
|------|----------|
| 单方面冷淡 | 3次主动无果后建议暂停 |
| 明确拒绝 | 尊重对方，停止追求 |
| 违反底线 | 立即终止并举报 |
| 长时间失联 | 7天无回应可标记放弃 |

### 4.5 用户行为规范

**用户行为准则**

| 规范类型 | 具体要求 |
|----------|----------|
| 真实性 | 资料信息需真实可靠，禁止虚构造假 |
| 尊重性 | 禁止骚扰、辱骂、威胁等行为 |
| 诚信性 | 禁止骗婚、骗财、传销等行为 |
| 隐私性 | 未经允许禁止泄露他人隐私 |
| 安全性 | 禁止线下交易、诱导投资等行为 |

**违规行为处罚**

| 违规级别 | 行为描述 | 处罚措施 |
|----------|----------|----------|
| 轻微 | 资料不完整、频繁换头像 | 提示完善 |
| 一般 | 频繁骚扰、虚假信息 | 限流7天 |
| 严重 | 诈骗、违法内容 | 永久封号 |
| 极其严重 | 违法犯罪 | 移交司法 |

## 五、应用场景

### 5.1 智能匹配系统

- **精准推荐**: 基于用户画像和偏好进行智能匹配
- **匹配度预测**: 预测匹配成功的可能性
- **最优推荐排序**: 根据多维度因素排序推荐列表

### 5.2 约会助手

- **开场白建议**: 根据对方资料生成个性化开场白
- **话题推荐**: 根据共同兴趣推荐聊天话题
- **约会策划**: 智能推荐约会场所和活动
- **时机提醒**: 提醒最佳发消息时间、约会邀请时机

### 5.3 关系顾问

- **关系诊断**: 分析关系状态和发展趋势
- **问题预警**: 识别潜在问题和风险
- **改进建议**: 提供关系改善建议
- **情感分析**: 分析双方互动中的情感变化

### 5.4 安全防护

- **身份认证**: 辅助验证用户真实身份
- **风险识别**: 识别诈骗倾向和不良用户
- **安全提醒**: 约会安全注意事项提醒
- **紧急求助**: 突发情况下的快速求助机制

## 六、核心指标体系

### 6.1 平台运营指标

| 指标名称 | 计算方式 | 目标值 |
|----------|----------|--------|
| 匹配成功率 | 匹配数 / 主动请求数 | ≥40% |
| 首次约会转化率 | 首次约会数 / 匹配数 | ≥30% |
| 关系确认率 | 确认关系数 / 首次约会数 | ≥25% |
| 用户7日留存 | 7日内活跃用户 / 新用户 | ≥45% |
| 付费转化率 | 付费用户 / 总用户 | ≥8% |

### 6.2 用户体验指标

| 指标名称 | 计算方式 | 目标值 |
|----------|----------|--------|
| 平均匹配耗时 | 首次匹配时间 - 注册时间 | ≤24h |
| 首次回复率 | 24h内回复数 / 匹配数 | ≥60% |
| 平均聊天轮次 | 总消息数 / 匹配对数 | ≥20轮 |
| 约会满意度 | 好评约会数 / 总约会数 | ≥80% |
| 关系持续率 | 3个月后仍交往 / 确认关系 | ≥50% |
