# 保险领域本体设计

## 概述

本文档定义保险领域的核心本体模型，用于指导知识库构建和智能化应用。涵盖保险产品、核保理赔、风险管理、客户服务等核心业务领域。

---

## 一、概念定义 (Concepts)

### 1.1 核心实体

| 概念 | 英文 | 定义 | 关键属性 |
|------|------|------|----------|
| 保险产品 | Insurance Product | 保险公司提供的风险保障商品 | 产品编码、名称、类型、保障范围、保险期限、保费 |
| 投保人 | Policyholder | 与保险公司签订保险合同的主体 | 客户编码、姓名/名称、证件类型、联系方式、职业 |
| 被保险人 | Insured | 享有保险保障权益的人 | 与投保人关系、健康状况、职业风险等级 |
| 受益人 | Beneficiary | 保险金受领人 | 与被保险人关系、受益顺序、受益份额 |
| 保单 | Policy | 保险合同的书面凭证 | 保单号、承保公司、产品类型、投保日期、生效日期 |
| 保费 | Premium | 投保人支付给保险公司的费用 | 保费金额、缴费方式（年缴/季缴/月缴）、缴费期限 |
| 保险金额 | Sum Insured | 保险公司承担的最高赔偿限额 | 基本保额、保额单位、是否可调整 |
| 理赔 | Claim | 保险事故发生后的赔偿申请 | 赔案号、出险日期、报案日期、赔付金额、案件状态 |
| 核保 | Underwriting | 保险公司评估风险并决定是否承保的过程 | 核保结论（标体/加费/除外/拒保）、核保评分 |
| 保险标的 | Insured Object | 保险合同保障的对象 | 标的类型、标的属性、价值评估 |
| 等待期 | Waiting Period | 保险责任生效前的时间限制 | 等待期天数、责任范围 |
| 免赔额 | Deductible | 保险人不承担赔偿的金额 | 免赔额类型（绝对/相对）、金额 |
| 现金价值 | Cash Value | 保单具有的累计价值 | 当前现金价值、退保金、贷款额度 |

## 二、规律 (Laws)

### 2.1 大数法则

**定义**: 大量随机现象的整体规律性，单个事件的不确定性在大量重复中将趋于稳定。

**保险应用**:
```
预期损失 = Σ(损失金额 × 发生概率) / n
```

**意义**:
- 保费厘定的基础：基于大量同类风险单位的损失数据进行精算
- 风险分散：通过承保大量同质风险降低不确定性
- 稳定性保证：承保数量越大，赔付率波动越小

### 2.2 保险费率厘定原则

**基本公式**:
```
毛保费 = 纯保费 + 附加保费
纯保费 = 预期损失 × 安全系数
附加保费 = 营业费用 + 预期利润 + 风险附加
```

**费率因子**:
| 因子类别 | 具体因素 |
|----------|----------|
| 风险因素 | 年龄、性别、健康状况、职业、使用性质 |
| 车辆因素 | 车龄、车型、排量、零整比 |
| 驾驶因素 | 驾龄、违章记录、出险次数 |
| 地域因素 | 地区、行驶区域、停车环境 |

### 2.3 损失率规律

**定义**: 实际赔付金额与保费收入的比率。

**计算公式**:
```
损失率 = 已决赔款 / 保费收入 × 100%
综合成本率 = 损失率 + 费用率
```

**行业基准**:
- 车险: 损失率约55-70%
- 医疗险: 损失率约60-80%
- 财产险: 损失率约40-60%
- 寿险: 损失率约20-40%

### 2.4 赔付率与保费调整规律

**规律**: 赔付率过高时需调整保费，过低时应回馈客户。

**调整触发线**:
| 综合成本率 | 应对措施 |
|------------|----------|
| < 95% | 盈利状态，可考虑优惠 |
| 95%-100% | 盈亏平衡，监控 |
| 100%-110% | 亏损，评估涨价 |
| > 110% | 严重亏损，必须调整 |

### 2.5 复效与现金价值规律

**复效规律**:
- 宽限期（通常60天）内未缴费，保单效力中止
- 两年内可申请复效，复效后需重新计算等待期
- 复效应补缴保费及利息

**现金价值规律**:
```
现金价值 = 已缴保费 × 现金价值比例 - 已领取金额
```
- 现金价值随缴费年限增长
- 前期现金价值较低，后期逐渐增加
- 退保损失通常在前几年较大

## 三、业务规则 (Rules)

### 3.1 核保规则

#### 核保等级评定

| 风险等级 | 定义 | 核保结论 | 费率调整 |
|----------|------|----------|----------|
| 标准体 | 正常风险 | 标体承保 | 标准费率 |
| 次标准体 | 高于正常风险 | 加费/除外 | +20%~100% |
| 拒保体 | 极高风险 | 拒保 | 不予承保 |
| 延期 | 暂时无法评估 | 延期承保 | 待定 |

#### 各险种核保要点

| 险种 | 必查项目 | 重点关注 | 常见除外 |
|------|----------|----------|----------|
| 寿险 | 财务状况、健康告知 | 既往病史、家族病史 | 艾滋病、毒品相关 |
| 医疗险 | 既往病史、体检 | 既往症、门诊记录 | 既往症、牙科、美容 |
| 重疾险 | 健康告知、BMI | 家族病史、吸烟史 | 先天性疾病、艾滋病 |
| 车险 | 车型、年限 | 出险记录、违章 | 改装部件、玻璃 |

### 3.2 承保规则

#### 最低/最高保额限制

| 险种 | 最低保额 | 最高保额 |
|------|----------|----------|
| 定期寿险 | 10万 | 1000万 |
| 重疾险 | 5万 | 100万 |
| 医疗险 | 1万 | 600万 |
| 意外险 | 10万 | 1000万 |
| 车损险 | 实际价值 | 新车购置价 |

#### 职业风险等级

| 等级 | 职业类别 | 承保政策 |
|------|----------|----------|
| 1类 | 办公室内勤 | 标准费率 |
| 2类 | 销售、教师 | 标准费率 |
| 3类 | 技术人员 | 轻度加费 |
| 4类 | 体力劳动者 | 中度加费 |
| 5类 | 高危职业 | 拒保或特殊费率 |
| 6类 | 极高危职业 | 拒保 |

### 3.3 理赔规则

#### 理赔时效要求

| 环节 | 时效要求 | 超时后果 |
|------|----------|----------|
| 报案时效 | 出险后48小时内 | 可能影响赔付 |
| 索赔时效 | 知道事故后2年内 | 丧失索赔权 |
| 理赔时效 | 资料齐全后30天内 | 超时支付利息 |
| 核定时效 | 情形复杂60天内 | 超时视为认可 |

#### 理赔所需材料（通用）

| 材料类型 | 具体内容 |
|----------|----------|
| 身份证明 | 身份证、户口本、营业执照 |
| 保险凭证 | 保单原件或电子保单 |
| 事故证明 | 事故认定书、死亡证明、伤残鉴定 |
| 医疗证明 | 诊断证明、住院小结、医疗发票 |
| 损失证明 | 维修发票、评估报告、照片 |

#### 理赔计算规则

**医疗险理赔**:
```
赔付金额 = (合理医疗费用 - 免赔额) × 赔付比例
```

**重疾险理赔**:
```
赔付金额 = 确诊即赔（一次性给付）
```

**车险理赔**:
```
车辆损失 = 实际修复费用（不超过保额）
第三者责任 = 实际损失（不超过保额）
```

### 3.4 保全规则

#### 保单贷款规则

| 项目 | 规则 |
|------|------|
| 贷款比例 | 不超过现金价值的80% |
| 贷款期限 | 最长6个月，可循环 |
| 贷款利率 | 保单约定利率（通常5-6%） |
| 逾期处理 | 利随本清，逾期超期保单失效 |

#### 保单变更规则

| 变更项目 | 规则 |
|----------|------|
| 受益人变更 | 被保险人同意，需书面申请 |
| 缴费方式变更 | 可以在缴费日前变更 |
| 保额调整 | 部分产品允许，需审核 |
| 联系方式变更 | 随时可变更 |

### 3.5 续保规则

| 续保类型 | 规则 |
|----------|------|
| 保证续保 | 无论健康状况变化，必须续保 |
| 承诺续保 | 审核后决定是否续保 |
| 续保审核 | 可能因历史赔付调整费率 |
| 宽限期 | 缴费日后60天为宽限期 |

## 五、应用场景

### 5.1 智能问答

基于本体知识可回答以下问题：
- "购买了医疗险，哪些情况可以理赔？"
- "重疾险的等待期是多长？"
- "保单贷款最多能贷多少？"
- "车险续保需要什么材料？"

### 5.2 智能核保

根据核保规则自动评估：
- 根据年龄、职业、健康状况给出核保结论
- 识别高风险因素并给出除外/加费建议
- 自动匹配产品与客户风险等级

### 5.3 理赔自动化

根据理赔规则实现：
- 自动识别理赔材料完整性
- 根据条款计算赔付金额
- 识别异常理赔模式，触发调查

### 5.4 产品推荐

基于客户画像和本体关系：
- 根据投保人/被保险人特征推荐产品
- 分析现有保单缺口，推荐保障补充
- 根据风险偏好推荐投资型产品

### 5.5 风险管理

基于规律和规则实现：
- 赔付率监控与预警
- 准备金充足性评估
- 再保险安排建议
