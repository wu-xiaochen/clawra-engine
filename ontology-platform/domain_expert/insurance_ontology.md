# 保险领域本体设计

## 概述

本文档定义保险领域的核心本体模型，用于指导知识库构建和智能化应用。涵盖保险产品、核保理赔、风险管理、客户服务、保险中介、再保险、监管合规等核心业务领域。本体与金融风控领域本体（finance_ontology.md）形成互补，本体侧重保险业务运营层面，风控本体侧重信用风险度量层面。

## 一、概念定义 (Concepts)

| 概念 | 英文 | 定义 | 关键属性 |

| 保险产品 | Insurance Product | 保险公司提供的风险保障商品 | 产品编码、名称、类型、保障范围、保险期限、保费 |
| 投保人 | Policyholder | 与保险公司签订保险合同的主体 | 客户编码、姓名/名称、证件类型、联系方式、职业 |

| 等待期 | Waiting Period | 保险责任生效前的时间限制 | 等待期天数、责任范围 |
| 免赔额 | Deductible | 保险人不承担赔偿的金额 | 免赔额类型（绝对/相对）、金额 |

## 二、规律 (Laws)

### 2.1 大数法则

**定义**: 大量随机现象的整体规律性，单个事件的不确定性在大量重复中将趋于稳定。

**保险应用**:
```
预期损失 = Σ(损失金额 × 发生概率) / n
```

**意义**:
- 保费厘定的基础：基于大量同类风险单位的损失数据进行精算
- 风险分散：通过承保大量同质风险降低不确定性
- 稳定性保证：承保数量越大，赔付率波动越小

### 2.2 保险费率厘定原则

**基本公式**:
```
毛保费 = 纯保费 + 附加保费
纯保费 = 预期损失 × 安全系数
附加保费 = 营业费用 + 预期利润 + 风险附加
```

**费率因子**:
| 因子类别 | 具体因素 |
|----------|----------|
| 风险因素 | 年龄、性别、健康状况、职业、使用性质 |
| 车辆因素 | 车龄、车型、排量、零整比 |
| 驾驶因素 | 驾龄、违章记录、出险次数 |
| 地域因素 | 地区、行驶区域、停车环境 |

### 2.3 损失率规律

**定义**: 实际赔付金额与保费收入的比率。

**计算公式**:
```
损失率 = 已决赔款 / 保费收入 × 100%
综合成本率 = 损失率 + 费用率
```

**行业基准**:
| 险种 | 损失率范围 | 综合成本率参考 |
|------|------------|----------------|
| 车险 | 55-70% | 95-105% |
| 医疗险 | 60-80% | 95-110% |
| 财产险 | 40-60% | 85-100% |
| 寿险 | 20-40% | 85-95% |
| 意外险 | 30-50% | 85-100% |

### 2.4 赔付率与保费调整规律

**规律**: 赔付率过高时需调整保费，过低时应回馈客户。

**调整触发线**:
| 综合成本率 | 应对措施 |
|------------|----------|
| < 95% | 盈利状态，可考虑优惠 |
| 95%-100% | 盈亏平衡，监控 |
| 100%-110% | 亏损，评估涨价 |
| > 110% | 严重亏损，必须调整 |

### 2.5 复效与现金价值规律

**复效规律**:
- 宽限期（通常60天）内未缴费，保单效力中止
- 两年内可申请复效，复效后需重新计算等待期
- 复效应补缴保费及利息

**现金价值规律**:
```
现金价值 = 已缴保费 × 现金价值比例 - 已领取金额
```
- 现金价值随缴费年限增长
- 前期现金价值较低，后期逐渐增加
- 退保损失通常在前几年较大

### 2.6 准备金规律

**定义**: 保险公司为履行保险责任而提取的资金储备。

**准备金类型**:
| 准备金类型 | 用途 | 提取依据 |
|------------|------|----------|
| 未到期责任准备金 | 承担未来保险责任 | 保费收入、风险期间 |
| 未决赔款准备金 | 已发生未赔付案件 | 报案数量、赔付预估 |
| 长期责任准备金 | 长期险种责任准备 | 精算模型、死亡率 |
| 保险保障基金 | 应对保险公司风险 | 保费一定比例 |

### 2.7 再保险分散规律

**分出规律**:
```
自留额 = 资本金 × 自留比例
分出额 = 风险敞口 - 自留额
```

**再保险安排原则**:
- 大数法则：通过再保险扩大风险分散面
- 责任分散：单一风险不超过资本金的一定比例
- 成本效益：再保费率应低于自留风险的预期损失

## 三、业务规则 (Rules)

### 3.1 核保规则

#### 核保等级评定

| 风险等级 | 定义 | 核保结论 | 费率调整 |
|----------|------|----------|----------|
| 标准体 | 正常风险 | 标体承保 | 标准费率 |
| 次标准体 | 高于正常风险 | 加费/除外 | +20%~100% |
| 拒保体 | 极高风险 | 拒保 | 不予承保 |
| 延期 | 暂时无法评估 | 延期承保 | 待定 |

#### 各险种核保要点

| 险种 | 必查项目 | 重点关注 | 常见除外 |
|------|----------|----------|----------|
| 寿险 | 财务状况、健康告知 | 既往病史、家族病史 | 艾滋病、毒品相关 |
| 医疗险 | 既往病史、体检 | 既往症、门诊记录 | 既往症、牙科、美容 |
| 重疾险 | 健康告知、BMI | 家族病史、吸烟史 | 先天性疾病、艾滋病 |
| 车险 | 车型、年限 | 出险记录、违章 | 改装部件、玻璃 |
| 意外险 | 职业类别 | 高风险运动、危险职业 | 战争、犯罪、酒驾 |

#### 健康告知规则

| 告知类型 | 告知内容 | 询问方式 |
|----------|----------|----------|
| 既往病史 | 过去5年住院、手术、诊断 | 列举式询问 |
| 家族病史 | 直系亲属重大疾病 | 列举式询问 |
| 投保史 | 既往投保、拒保、延期 | 累计保额询问 |
| 健康状况 | 目前健康、烟酒习惯 | 概括式询问 |
| 职业风险 | 工作内容、高风险因素 | 职业分类询问 |

### 3.3 理赔规则

#### 理赔时效要求

| 环节 | 时效要求 | 超时后果 |
|------|----------|----------|
| 报案时效 | 出险后48小时内 | 可能影响赔付 |
| 索赔时效 | 知道事故后2年内 | 丧失索赔权 |
| 理赔时效 | 资料齐全后30天内 | 超时支付利息 |
| 核定时效 | 情形复杂60天内 | 超时视为认可 |
| 诉讼时效 | 知道权益受损3年内 | 丧失胜诉权 |

#### 理赔所需材料（通用）

| 材料类型 | 具体内容 |
|----------|----------|
| 身份证明 | 身份证、户口本、营业执照 |
| 保险凭证 | 保单原件或电子保单 |
| 事故证明 | 事故认定书、死亡证明、伤残鉴定 |
| 医疗证明 | 诊断证明、住院小结、医疗发票 |
| 损失证明 | 维修发票、评估报告、照片 |

#### 理赔计算规则

**医疗险理赔**:
```
赔付金额 = (合理医疗费用 - 免赔额) × 赔付比例
```
- 社保先赔：社保报销后剩余部分×100%赔付
- 未经社保：医疗费用×60%赔付

**重疾险理赔**:
```
赔付金额 = 确诊即赔（一次性给付）
```
- 确诊合同约定疾病：100%保额
- 达到约定状态：100%保额
- 实施约定手术：100%保额

**车险理赔**:
```
车辆损失 = min(实际修复费用, 保额) - 免赔额
第三者责任 = min(实际损失, 保额) - 免赔额
```

**寿险理赔**:
```
身故金 = 保额（扣除已领取年金）
全残金 = 保额
```

#### 理赔调查规则

| 调查触发条件 | 调查内容 | 调查方式 |
|--------------|----------|----------|
| 重大案件（超10万） | 事故真实性、是否存在除外 | 面访、取证 |
| 异常索赔 | 既往病史、道德风险 | 社保记录、医院调查 |
| 短时间内出险 | 逆选择可能 | 详细取证 |
| 伤残鉴定存疑 | 伤残等级真实性 | 重新鉴定 |

### 3.4 保全规则

#### 保单贷款规则

| 项目 | 规则 |
|------|------|
| 贷款比例 | 不超过现金价值的80% |
| 贷款期限 | 最长6个月，可循环 |
| 贷款利率 | 保单约定利率（通常5-6%） |
| 逾期处理 | 利随本清，逾期超期保单失效 |
| 贷款期间保障 | 贷款期间保障责任继续有效 |

#### 保单变更规则

| 变更项目 | 规则 | 生效时间 |
|----------|------|----------|
| 受益人变更 | 被保险人同意，需书面申请 | 即时生效 |
| 缴费方式变更 | 可以在缴费日前变更 | 下期生效 |
| 保额调整 | 部分产品允许，需审核 | 审核通过后 |
| 联系方式变更 | 随时可变更 | 即时生效 |
| 职业变更 | 高风险需加费或拒保 | 即时生效 |
| 续期账户变更 | 银行账户变更 | 下期扣款生效 |

### 3.5 续保规则

| 续保类型 | 规则 | 费率调整 |
|----------|------|----------|
| 保证续保 | 无论健康状况变化，必须续保 | 不因个人理赔调整 |
| 承诺续保 | 审核后决定是否续保 | 可调整费率 |
| 续保审核 | 可能因历史赔付调整费率 | 整体调整 |
| 宽限期 | 缴费日后60天为宽限期 | 宽限期后失效 |

### 3.6 保险监管规则

#### 偿付能力监管

| 指标 | 要求 | 监管红线 |
|------|------|----------|
| 核心偿付能力充足率 | ≥50% | <30% 监管谈话 |
| 综合偿付能力充足率 | ≥100% | <50% 限制业务 |
| 风险综合评级 | B类以上 | D类限制投资 |

#### 资金运用规则

## 六、关键指标

### 6.1 经营指标

| 指标 | 计算公式 | 正常范围 |
|------|----------|----------|
| 原保险保费收入 | 保费收入 | - |
| 已赚保费 | 保费收入 - 未到期责任准备金变化 | - |
| 赔付率 | 已决赔款 / 已赚保费 × 100% | 40-70% |
| 费用率 | 营业费用 / 已赚保费 × 100% | 20-35% |
| 综合成本率 | 赔付率 + 费用率 | 85-105% |
| 投资收益率 | 投资收益 / 资金运用规模 | 3-6% |

### 6.2 偿付能力指标

| 指标 | 计算公式 | 监管要求 |
|------|----------|----------|
| 核心偿付能力充足率 | 核心资本 / 最低资本 × 100% | ≥50% |
| 综合偿付能力充足率 | 实际资本 / 最低资本 × 100% | ≥100% |
| 保险保障基金 | 保费 × 一定比例 | 足够应对风险 |

### 6.3 客户指标

| 指标 | 定义 | 业务意义 |
|------|------|----------|
| 13个月继续率 | 13个月后仍在缴费的保单比例 | 客户满意度 |
| 25个月继续率 | 25个月后仍在缴费的保单比例 | 长期客户价值 |
| 出险率 | 已承保保单出险比例 | 风险评估准确性 |
| 理赔时效 | 立案到结案平均天数 | 服务效率 |
